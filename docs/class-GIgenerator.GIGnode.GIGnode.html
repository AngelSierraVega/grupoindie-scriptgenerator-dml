Class GIGnode

Represents the main element of a descriptive markup languaje such as
LaTeX, XML and HTML (see https://en.wikipedia.org/wiki/Markup_language
for more information).

Example:

Simple node

echo new GIGnode("node");

<node></node>


Example:

Open node

echo new GIGnode("node",true);

<node>


Example:

Node with simple attribue

echo new GIGnode("node",false,["attr"]);

<node attr></node>


Example:

Node with attribue-value

echo new GIGnode("node",false,["attr"=>"value"]);

<node attr='value'></node>


Example:

Simple node with content

echo new GIGnode("node",false,[],["content of the node"]);

<node>content of the node</node>


Example:

Parent-child nodes

echo new GIGnode("parent",false,[],[new GIGnode("child")]);

<parent><child></child></parent>
