Class Factory

Implements a Factory Pattern for a Descriptive Markup Languaje (DML).

LaTeX, XML, and HTML are examples of languajes that can be generated using
this class (more info. at https://en.wikipedia.org/wiki/Markup_language).

GIndie\DML\Node\Factory uses GIndie\DML\Node\_protectedAttrs, GIndie\DML\Node\_publicAttrs, GIndie\DML\Node\_presentationSemantics
